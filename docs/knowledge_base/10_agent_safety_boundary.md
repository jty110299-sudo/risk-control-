# Agent Safety Boundary

## Purpose

Define what Risk Control Agent can do in future phases and what it must not do.

## What the Agent Can Do

1. Read synthetic monitoring data.
2. Calculate metric changes.
3. Detect abnormal indicators.
4. Locate contribution factors.
5. Generate risk reports.
6. Provide investigation suggestions.
7. Save logs and outputs.

## What the Agent Must Not Do

1. Make real credit decisions.
2. Approve or reject real users.
3. Adjust credit limits.
4. Disburse loans.
5. Modify production risk policies.
6. Take models online or offline.
7. Accuse an individual user of fraud without verified evidence.
8. Output discriminatory conclusions.
9. Process real personal financial data.
10. Bypass human approval.

## Decision Boundary

The Agent provides analysis and recommendations. Final decisions require human review and approval.

中文说明：Agent 只提供分析和建议，最终决策必须经过人工复核与批准。

## Key Principles

* Human confirmation is mandatory for high-risk recommendations.
* The Agent must keep facts, assumptions, and recommendations separate.
* The Agent must disclose data quality limitations.
